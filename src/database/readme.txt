Just Journal currently runs on MySQL 4.1.x.  Included in this directory
are sql files to create the tables needed to run JJ.

MySQL Version Compatibility
MySQL 4.0 and lower will not work with Just Journal.  MySQL 5.0.x is untested
at this time.

JDBC Driver
The production version of just journal is currently running on mysql-connector-j
3.1.10.  3.1.8 is also known to work.
